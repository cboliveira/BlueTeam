# 🌐 Network Security

Scripts, tools, and research focused on network analysis and security assessment.

## Contents

| File/Folder | Description |
|-------------|-------------|
| `port-scanner.py` | Custom TCP/UDP port scanner with banner grabbing |
| `subnet-enum.py` | Subnet enumeration and host discovery |
| `packet-analyzer/` | Wireshark filters and packet capture analysis |
| `nmap-scripts/` | Custom NSE scripts and scan profiles |

## Tools Used

- **Nmap** — Host discovery and port scanning
- **Wireshark / tshark** — Packet analysis
- **Scapy** — Packet crafting and manipulation
- **Netcat** — Network debugging and banner grabbing

## Key Concepts Covered

- TCP/IP fundamentals and packet analysis
- Network enumeration methodology
- Service version detection
- Traffic analysis for anomaly detection

> All scans performed on personal lab or authorized targets only.
